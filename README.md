# MAHMUTGOKSU.github.io

Personal academic homepage of Mahmut Zahid Göksu.

Built with [Jekyll](https://jekyllrb.com/), based on [RayeRen/acad-homepage.github.io](https://github.com/RayeRen/acad-homepage.github.io) (MIT), itself a fork of [academicpages](https://github.com/academicpages/academicpages.github.io) / [minimal-mistakes](https://github.com/mmistakes/minimal-mistakes).

The previous Next.js version is archived on branch `archive/nextjs`.

## Local preview

```bash
bundle install
bundle exec jekyll serve
# open http://localhost:4000
```

Content edits live in `_pages/about.md` and `_config.yml`.
